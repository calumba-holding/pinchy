# Sample Documents

These files are mounted into the Pinchy container at `/data/sample-docs/`
for development and testing purposes.

`sample-product-handbook.pdf` is fictional demo content (a fake kettle
product handbook) for exercising the knowledge-base agent's PDF ingest and
retrieval flow — see [Create a Knowledge Base Agent](https://docs.heypinchy.com/guides/create-knowledge-base-agent/).
